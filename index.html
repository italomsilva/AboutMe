Projetos

Air Jordan 1

Este projeto é uma landing page dedicada ao icônico tênis Air Jordan 1. Desenvolvido inteiramente do zero, cada aspecto foi cuidadosamente criado, desde a seleção da paleta de cores, construção do layout, funcionalidades implementadas até as imagens utilizadas foram editadas manualmente, garantindo um visual exclusivo e profissional. Cada detalhe foi pensado para proporcionar uma experiência visual impactante e envolvente para os fãs do Air Jordan.

Tecnologias utilizadas:

  • Css

  • JavaScript

  • React
